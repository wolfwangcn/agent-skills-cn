# 每日总结 (Daily Summary)

生成当天活动的结构化摘要。

## 指令

回顾今天的所有对话和活动，提炼出结构化的每日总结。重点是**行动项和决策**，不是流水账。

## 输出格式

```markdown
# [YYYY-MM-DD] 日报

## 📌 关键决策
- [今天做的重要决定，以及为什么]

## ✅ 完成
- [做完的事情]

## 🔄 进行中
- [开始但未完成的事]
- 卡在哪：[如果有卡点]

## 💡 收获
- [值得记住的洞察、教训、发现]

## 📋 明天
- [ ] [具体的待办事项]
```

## 规则

- 每项一行，简洁有力
- 只记有价值的内容 — "回复了3条消息"不值得记
- 如果今天比较轻松：简短说明即可，不要凑字数
- 关键决策必须记录**原因**（"选了 A 因为..."）
- 存入 `memory/YYYY-MM-DD.md`
- 明天的待办要具体可执行（"写完 X 的第一版" 而不是 "继续搞 X"）

## 触发时机

- 用户主动要求时
- 一天结束时（如在 heartbeat 中检测到大量活动）
- 项目里程碑完成后
