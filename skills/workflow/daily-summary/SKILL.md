# 每日总结 (Daily Summary)

## 描述
在一天结束时，自动生成当日工作/学习/对话的结构化摘要，存入记忆文件。

## 触发条件
- 用户说"今天做了什么"、"总结一下"、"日报"
- 或在 heartbeat 中检测到当天有大量活动

## 输出格式

```markdown
# [YYYY-MM-DD] 日报

## 📌 重要事项
- [最重要的 1-3 件事]

## ✅ 完成
- [做完的事情列表]

## 🔄 进行中
- [还没做完的事情]

## 💡 发现/想法
- [值得记住的洞察、想法]

## 📋 明日待办
- [建议明天做的事]
```

## 规则
- 简洁为主，每项一行
- 重要的事放前面
- 只记有价值的内容，不记流水账
- 如果用户没特别做什么，就简短说"今天比较轻松，没有重大事项"
- 存入 `memory/YYYY-MM-DD.md`
