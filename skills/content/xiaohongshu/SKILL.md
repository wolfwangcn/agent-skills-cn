# 小红书文案 (Xiaohongshu)

基于小红书 CES 评分模型、推荐算法机制和内容权重因素，生成高互动率笔记。

---

## 平台算法深度理解

### CES 评分模型（Community Engagement Score）

小红书用 CES 分数决定笔记是否进入下一级流量池：

```
CES = 点赞(1分) + 收藏(1分) + 评论(4分) + 转发(4分) + 关注(8分)
```

**核心洞察**：评论和转发权重是点赞的 4 倍，关注是 8 倍。所以：
- 写能引发讨论的内容（>收藏型干货）
- 设置互动钩子（问题/投票/求推荐）
- 引导关注比引导点赞重要 8 倍

### 流量分发机制

```
发布 → 审核（10min-2h）
  ↓
小流量池测试（200-500 曝光）
  ↓ CES 达标？
中流量池（2000-5000 曝光）
  ↓ CES 持续达标？
大流量池（1w-10w+ 曝光）
  ↓
搜索长尾（持续数月获取搜索流量）
```

**关键窗口**：发布后前 2 小时的互动数据决定是否进入下一级流量池。

### 笔记权重因素

| 因素 | 权重 | 具体指标 |
|------|------|---------|
| 账号权重 | 25% | 等级、垂直度、更新频率、违规历史、粉丝真实度 |
| 内容质量 | 30% | 原创度、信息密度、图片清晰度、文案完整度 |
| 互动率 | 35% | 前2h的CES分数、互动率(互动/曝光) |
| 标签匹配 | 10% | 话题标签精准度、内容与标签一致性 |

### 搜索 SEO 因素

小红书已成为年轻人的"搜索引擎"。搜索流量占总流量 30-40%：
- 标题含关键词（用户会搜什么？）
- 正文前 100 字含核心关键词
- 标签与搜索词匹配
- 评论区出现关键词也计入

## 指令

帮用户生成符合小红书算法偏好的高互动笔记。严格按以下流程。

### 第一步：确认要素

1. **内容类型**：
   - 好物推荐（种草/测评）
   - 避坑指南（踩雷/对比）
   - 教程攻略（步骤/方法）
   - 经验分享（职场/学习/生活）
   - 探店打卡（美食/旅行）
   - 变美记录（穿搭/护肤/健身）

2. **核心信息**：推荐什么？卖点？价格？效果？

3. **目标受众**：年龄段？职业？场景？

4. **目标关键词**：用户会搜什么找到这篇？（影响 SEO）

### 第二步：封面设计建议

**尺寸**：3:4 竖版（1080×1440px）— 信息流中最大展示面积

**封面类型与 CTR 数据**：

| 类型 | 点击率对比 | 适合 |
|------|-----------|------|
| 真人出镜 | 基准 +30% | 穿搭、护肤、探店 |
| 对比图（before/after） | 基准 +45% | 减肥、改造、测评 |
| 数字大标题 | 基准 +25% | 攻略、清单、干货 |
| 产品平铺 | 基准 +15% | 好物推荐、开箱 |

**设计规则**：
- 文字占封面 30-50%（太少没信息，太多影响美观）
- 色系：高饱和暖色（红/橙/黄）CTR 比冷色高 15-20%
- 核心信息放上半部分（信息流中下半部分可能被裁切）
- 字体：加粗无衬线体，描边或阴影增加可读性

### 第三步：标题撰写

**标题公式**（数据验证有效）：

| 公式 | 结构 | 示例 | CTR提升 |
|------|------|------|--------|
| 数字+结果 | N天/款/招+[效果] | "7天瘦5斤的懒人食谱" | +35% |
| 身份+痛点 | [人群]+[解决方案] | "学生党平价护肤全套不过百" | +25% |
| 反差/悬念 | [意外]+[结果] | "被柜姐嫌弃后我自学了化妆" | +40% |
| 合集/清单 | [数量]+[类别]+[评价] | "8家必吃小吃，本地人推荐" | +20% |
| 时间限定 | [年份/季节]+[推荐] | "2024秋冬必入的5件外套" | +15% |

**标题规则**：
- 字数：15-25字（超过在搜索结果中被截断）
- 必须含核心关键词（SEO）
- 至少含一个：数字 / emoji / 对比词
- 给用户 3 个备选

### 第四步：正文撰写

**正文结构模板**：

```
[Hook - 1句话痛点/结论前置]

[正文主体 - 分点阐述]
📍/💡/✨ 要点1：[描述]
📍/💡/✨ 要点2：[描述]
📍/💡/✨ 要点3：[描述]

[真实感受 - 优缺点都说]
✅ 喜欢的点：...
❌ 不完美的：...

[互动钩子 - 引发评论]
你们觉得呢？/有什么想问的评论区聊！
```

**风格特征**：
- 第一人称，像和朋友聊天
- 每段 2-3 个 emoji（不能没有，也不能满屏都是）
- 短句为主（句号多于逗号）
- 感叹号表达真实情感（但不要每句都感叹）
- 有"人设感"：不是广告，是真实分享
- 适当口语化："绝了"、"真的会谢"、"谁懂"

**字数**：300-800字（平台最佳阅读完成率区间）

**互动钩子设计**（CES 权重最高项）：
- 提问式："你们平时用什么？评论区交流！"
- 求助式："还有什么好用的求推荐！"
- 投票式："A还是B？选A的扣1选B的扣2"
- 争议式："我觉得X比Y好用，你们同意吗？"

### 第五步：标签策略

**组合公式**（8-12个标签）：

```
2个 热门大标签（>1亿浏览量）— 获取曝光
+ 3个 中等标签（1000万-1亿）— 精准分发
+ 3-5个 长尾标签（<1000万）— 搜索长尾
+ 1-2个 品牌/产品标签 — 精准流量
```

**标签查找方法**：
- 搜索框输入关键词看下拉联想
- 看同类爆款笔记用了什么标签
- 千瓜数据 https://www.qian-gua.com/ 查热门标签

### 第六步：发布时间

| 时段 | 适合品类 | 竞争度 |
|------|---------|--------|
| 7:00-9:00 | 职场干货、早餐 | 中 |
| 12:00-14:00 | 美食、穿搭 | 高 |
| 17:00-18:00 | 下班、美食 | 中 |
| 20:00-22:00 | 护肤、情感、好物 | 最高（流量也最大） |
| 22:00-23:30 | 深度分享、情感 | 低（性价比高） |

## 反面案例

❌ 通篇夸赞无缺点（"这个真的太好用了必须入！"— 像广告）
✅ 优缺都说（"唯一不满意的是包装太大不方便携带，但效果确实惊艳"）

❌ 没有 emoji 的纯文字长段落（不符合平台调性）
✅ 短句 + emoji 节奏（"试了一周✨ 效果真的有！但也不是那种立竿见影"）

❌ 标签全用大词（#好物推荐 #分享 — 太泛，竞争激烈）
✅ 大中小搭配（#好物推荐 #油皮护肤 #学生党水乳推荐 #珂润面霜测评）
