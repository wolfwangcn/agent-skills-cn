# 小红书文案 (Xiaohongshu)

## 描述
生成符合小红书平台风格的种草/分享/攻略文案，适合直接发布。

## 触发条件
用户提到"小红书"、"种草"、"笔记"

## 风格特征
- 第一人称，亲和力强
- 大量 emoji（每段 2-4 个）
- 短句，节奏快
- 感叹号多！表达真实感受！
- 口语化，像在跟闺蜜/朋友聊天

## 输出格式

```
[封面标题建议 - 10字以内，带emoji]

[正文]
开头：一句话痛点/惊喜 hook
中间：分点列出核心信息（用 emoji bullet）
结尾：总结推荐/踩雷

[标签]
#话题标签1 #话题标签2 #话题标签3 ...（5-10个）
```

## 流程

### 第一步：确认类型
- 好物推荐 / 踩雷避坑
- 教程攻略 / 经验分享
- 探店打卡 / 旅行记录
- 职场干货 / 学习方法

### 第二步：确认信息
- 推荐什么？核心卖点？
- 价格区间？
- 适合什么人？
- 有对比参照吗？

### 第三步：生成文案
按上述格式输出，同时给出：
- 封面图建议（构图、文字）
- 发布时间建议

## 规则
- 标题必须有数字或 emoji
- 正文 300-800 字（小红书最佳阅读长度）
- 不能有广告感太重的词（"强烈推荐"可以，"赶紧下单"不行）
- 标签选热门但相关的
- 给 2 个标题备选
