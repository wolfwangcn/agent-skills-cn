# 公众号写作 (WeChat Article)

## 描述
按照微信公众号的阅读习惯和排版风格生成文章，适合直接发布。

## 触发条件
用户提到"公众号"、"微信文章"、"写篇推文"

## 流程

### 第一步：确认要素
1. **主题**：写什么？
2. **受众**：给谁看？（技术人/白领/学生/大众）
3. **风格**：干货型 / 故事型 / 观点型 / 盘点型
4. **字数**：短文(800-1500) / 中文(1500-3000) / 长文(3000+)

### 第二步：生成结构
```
标题（吸引点击但不标题党）
├── 开头（痛点/故事/提问，3 句话内抓住注意力）
├── 正文
│   ├── 核心观点 1 + 论证/案例
│   ├── 核心观点 2 + 论证/案例
│   └── 核心观点 3 + 论证/案例
├── 金句/总结（适合截图转发的那种）
└── 结尾（互动引导：点赞/在看/留言）
```

### 第三步：排版规则
- 段落短：每段 2-4 行
- 多用空行分隔
- 重点**加粗**
- 适当用 emoji 但不过度
- 小标题用「」或 ▎ 前缀
- 不用 Markdown 标题语法（公众号不支持）

## 规则
- 口语化 > 书面化
- 一个段落只说一件事
- 开头 3 行决定读者走不走——不要废话开场
- 标题给 3 个备选
- 结尾不要"感谢阅读"这种废话
